═══════════════════════════════════════════════════════════════════
  KURZ-ANLEITUNG (wenn es schnell gehen muss)
═══════════════════════════════════════════════════════════════════

1. Node.js installieren: https://nodejs.org (LTS Version)
   → Computer NEU STARTEN!

2. PowerShell in diesem Ordner öffnen (Rechtsklick → "In Terminal öffnen")

3. Eingeben:
   npm install
   npm start

4. Im Browser testen: http://localhost:3001/api/health

Fertig! ✅
