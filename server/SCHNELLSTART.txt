═══════════════════════════════════════════════════════════════════
  ERSTMALIGE INSTALLATION
═══════════════════════════════════════════════════════════════════

1. Node.js installieren von: https://nodejs.org
   → LTS Version (grüner Button)
   → Computer neu starten!

2. PowerShell in diesem Ordner öffnen:
   → Rechtsklick auf "server" Ordner
   → "In Terminal öffnen"

3. Dependencies installieren:
   
   npm install

4. Server starten:
   
   npm start


═══════════════════════════════════════════════════════════════════
  TÄGLICHE NUTZUNG (nach Installation)
═══════════════════════════════════════════════════════════════════

Server starten:

   npm start

Server stoppen:

   Strg + C  (im PowerShell-Fenster)


═══════════════════════════════════════════════════════════════════
  TESTEN OB ES FUNKTIONIERT
═══════════════════════════════════════════════════════════════════

Im Browser öffnen:

   http://localhost:3001/api/health

Erwartete Ausgabe:

   {"status":"ok","message":"Moodboard Server is running"}


═══════════════════════════════════════════════════════════════════
  NETZWERK-IP FINDEN (für Handy-Zugriff)
═══════════════════════════════════════════════════════════════════

In PowerShell ausführen:

   ipconfig

Suche nach: "IPv4-Adresse"
Beispiel: 192.168.1.100

Diese IP brauchst du für:
- Handy-Zugriff
- Tablet-Zugriff  
- Andere Computer im Netzwerk


═══════════════════════════════════════════════════════════════════
  FIREWALL ÖFFNEN (für Handy/Tablet-Zugriff)
═══════════════════════════════════════════════════════════════════

PowerShell als Administrator öffnen:

   Rechtsklick auf PowerShell → "Als Administrator ausführen"

Dann ausführen:

   New-NetFirewallRule -DisplayName "Moodboard Server" -Direction Inbound -LocalPort 3001 -Protocol TCP -Action Allow


═══════════════════════════════════════════════════════════════════
  HANDY-ZUGRIFF TESTEN
═══════════════════════════════════════════════════════════════════

1. Handy mit gleichem WLAN verbinden
2. Browser auf Handy öffnen
3. Gehe zu: http://192.168.1.100:3001/api/health
   (ersetze 192.168.1.100 mit deiner IP von oben)

Sollte anzeigen: {"status":"ok",...}

═══════════════════════════════════════════════════════════════════
  HÄUFIGE PROBLEME
═══════════════════════════════════════════════════════════════════

Problem: "npm: Die Benennung wurde nicht als Name eines Cmdlet erkannt"
Lösung: Node.js installieren und Computer neu starten

Problem: "Port 3001 already in use"  
Lösung: Port in src/server.js ändern auf 3002

Problem: Handy kann nicht zugreifen
Lösung: 
  1. Firewall-Regel erstellen (siehe oben)
  2. Beide Geräte im gleichen WLAN?
  3. Richtige IP verwendet?

Problem: Server stoppt wenn ich PowerShell schließe
Lösung: PowerShell-Fenster offen lassen ODER automatischen Start einrichten
